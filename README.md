任务三：

1、芯片手册阅读：学习了如何快速定位关键信息，包括引脚定义、电气特性（电平范围、功耗）、封装尺寸以及最核心的参考应用电路等；理解了芯片稳定运行的“三要素”：电源滤波、时钟源、复位与通信接口。

2、I2C与SPI通信协议：【提交了I2C读取MPU6050的工程产出】
    I2C协议（MPU6050）：半双工通信，采用主从架构。核心在于从机地址寻址、起始/停止信号以及ACK应答机制。电路简单，速率相对较低。
    SPI协议（ICM20602）：全双工通信，拥有独立的发送（MOSI）和接收（MISO）线路。通过CS（片选信号）指定目标器件，时钟频率较高。

3、原始数据处理：
    数据解算：IMU输出的通常是16位的原始补码（Hex）。需要根据手册中的量程将其转换为实际的加速度（g）和角速度（deg/s）。
    坐标系变换：通过数学模型将传感器坐标系映射到机器人坐标系。
    姿态表示：理解了欧拉角（Pitch俯仰、Roll横滚、Yaw偏航）的物理含义，并简单了解了消除陀螺仪漂移的融合算法思路（如互补滤波/卡尔曼滤波）。

4、外骨骼“零力模式”的学习：
    零力模式：通过控制算法抵消机械结构自身的惯性、摩擦力和重力，使穿戴者几乎感知不到机械臂的存在。
    电流环控制：电机电流与输出力矩成正比。通过高频采样电机端的电流/力矩传感器信号，实时调整PWM占空比，使电机产生的力矩刚好等于机械结构的阻力矩。
    意义：零力控制是“追随控制”的基础，只有实现了精准的电流环控制，外骨骼才能实现灵敏的“主动感知”和“意图跟随”。

任务四：

1、焊接工艺理论：
    拖焊：学习了针对高密度贴片芯片（如QFP封装的STM32）的焊接技巧。通过助焊剂辅助，利用焊锡的表面张力一次性完成整排引脚的焊接，可以极大提高效率并减少桥连风险。
    堆焊：针对大电流接口（如XT60电源口或大功率电机驱动接口），学习了通过堆叠焊锡或加焊铜丝来增强导电截面积的方法，以承载爆发力矩时产生的大电流。
    焊点质量检查：学会了判断“冷焊”、“虚焊”与“圆润焊点”，确保在剧烈振动环境下（如行走时）电路的稳定性。

2、系统级设计思维与拓扑优化：
    软硬结合设计：了解了硬件布局如何影响软件算法的实时性。例如，将IMU尽量靠近转动轴心以减少离心加速度对姿态解算的影响……
    连线优化：通过合理排布接口位置，减少物理线束的交叉与冗余，降低系统受电磁干扰的风险。
    重心与散热：考虑到外骨骼的动态响应，在布板时应优化重负载器件的位置，并预留功率器件的散热铜皮。

3、PID控制算法的离散化实现：
    离散化思维：理计算机系统是定采样周期运行的，需要将连续的积分变为累加，将微分变为差分。
    位置式PID：直接输出目标电流，适用于需要绝对力矩控制的场景。
    增量式PID：输出电流的改变量，具有更强的抗干扰能力，且在系统切换或断电重启时更加安全。
    也了解了离散化PID的基本公式。
